This cheat sheet covers the most useful CSS Grid properties with examples, explanations, and code snippets for quick reference.

1. Basic Grid (3x3)

grid-template-columns and grid-template-rows define how many columns and rows the grid should have.

2

.grid-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-template-rows: repeat(3, 100px);
}

3. Grid with Gaps

The gap property sets spacing between cells.

1
2
3
4
5
6
7
8
9
.grid-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 60px;
}

4. Grid Item Placement

Items can span multiple rows/columns using grid-column and grid-row.

1
2
3
4
5
.box:first-child {
  grid-column: 1 / 3;
  grid-row: 1 / 4;
}
.box:last-child {
  grid-column: 1 / 3;
}

5. Grid Template Areas

Named grid areas make layouts easier to manage.

Header
Sidebar
Main
Footer
.grid-container {
  display: grid;
  grid-template-areas:
    "header header header"
    "sidebar main main"
    "footer footer footer";
}
.box.header { grid-area: header; }
.box.sidebar { grid-area: sidebar; }
.box.main { grid-area: main; }
.box.footer { grid-area: footer; }

6. Auto-Fit and Minmax

Responsive grid using auto-fit and minmax.

.grid-container {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(100px, 1fr));
}

7. Alignment and Justification

justify-content controls horizontal alignment, align-items controls vertical.

1
2
3
.grid-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  height: 300px;
  justify-content: space-between;
  align-items: center;
}
